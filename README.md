# memristor-sde-diffusion

Code for "Resistive memory-based neural differential equation solver for score-based diffusion model"

## Abstract

While AI-Generated Content (AIGC) strives to replicate human imagination, current models like score-based diffusion remain slow and energy-intensive. This inefficiency stems from conventional digital computers, where physically separated storage and processing units cause data-transfer bottlenecks, and discrete operations disrupt naturally continuous generation dynamics. Here we show a brain-inspired, analog in-memory computing system that overcomes these limitations. By employing resistive memory, our system seamlessly integrates storage and computation to act as a time-continuous neural differential equation solver. We experimentally validate our solution with 180 nm resistive memory in-memory computing macros. Demonstrating equivalent generative quality to the software baseline, our system achieved enhancements in generative speed for both unconditional and conditional generation tasks, by factors of 69.0 and 116.5, respectively. Moreover, it accomplished reductions in energy consumption of 31.5% and 52.0%. Our approach expands the horizon for hardware solutions in edge computing for generative AI applications.

## Overview

This repository contains the research code associated with the paper, including:

- Python notebooks and helper modules for unconditional and conditional score-based diffusion experiments.
- An MCU project for the hardware demonstration.

This README focuses on the software workflow outside the MCU project.

## Python Components

### `conditional/`

- `VAE_v1.ipynb` trains a compact VAE on [EMNIST](https://www.nist.gov/itl/products-and-services/emnist-dataset), an open public dataset, and exports a low-dimensional latent dataset.
- `HDPM.ipynb` trains and samples a conditional diffusion model in the latent space.
- `models.py` defines the MLP score networks, VAE variants, and related modules.
- `my_sde.py` implements SDE training and sampling routines.
- `my_utils.py` provides dataset helpers, embeddings, perturbation utilities, and evaluation helpers.

### `unconditional/`

- `HDPM copy.ipynb` contains the unconditional 2D diffusion example.
- `my_sde.py` contains the unconditional score model and SDE-based training and sampling code.

## Recommended Workflow

### Conditional Workflow

1. Open `conditional/VAE_v1.ipynb`.
2. Download [EMNIST](https://www.nist.gov/itl/products-and-services/emnist-dataset), an open public dataset, through the notebook and train the VAE.
3. Export the latent dataset generated by the notebook.
4. Open `conditional/HDPM.ipynb`.
5. Train the conditional score model on the latent samples.
6. Run the sampling cells to generate class-conditioned points.

### Unconditional Workflow

1. Open `unconditional/HDPM copy.ipynb`.
2. Generate the synthetic 2D dataset.
3. Train the score network.
4. Run the sampling cells to visualize generated samples.

## Outputs

- The notebooks generate intermediate files such as trained model checkpoints and exported latent datasets.
- Notebook paths are relative to their own directories.
